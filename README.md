# Welcome to the Selaco Source Code!

Selaco runs on a lightly modified version of GZDoom. 
https://github.com/ZDoom/gzdoom/

Big thanks goes out to the talented folks that spent many years getting GZDoom where it is today!

## How to build Selaco

To build Selaco, follow the instructions for GZDoom. See the [wiki](https://zdoom.org/wiki/) and see the "Programmer's Corner" on the bottom-right corner of the page to build for your platform.
